Grid Definition

One of the major difference of grid over flex is grid is a 2D design layout because we can position owr elements in by using both the grid column & grid row as well

grid-column-end:-1 like -ve value properties can help you to create dynamic end position of grid column/rows

If grid-column-start:4 is higher than grid-column-end:2 then values will be flipped (i.e g-c-s:2,g-c-e:4) automatically.

If you don't specify g-c-s & g-c-e property then the default / initial value 'auto' will be applied automatically.

Note: span property in grid doesn't support -ve values.
